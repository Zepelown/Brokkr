# Brokkr

Brokkr is a NeoForge weapon enhancement mod for Minecraft 1.21.6.

## Features

- Enhance supported weapons up to +20.
- Each enhancement level adds +0.2 attack damage.
- Tiered enhancement success chances.
- Enhancement result messages are localized.
- Enhanced weapons show level prefixes and hit particles.
